用於事件驅動應用程式的低代碼編程

收集、轉換和視覺化即時數據的最簡單方法。

最新版本：v4.1.7 (npm)

開始使用

Node-RED 的目標是讓任何人都能建立收集、轉換和視覺化數據的應用程式；建立可以自動化其世界的流程（Flow）。其低代碼的特性使其適用於任何背景的使用者，無論是用於家庭自動化、工業控制系統或兩者之間的任何應用。

從家庭愛好者到大規模工業運作，Node-RED 已在廣泛的應用中佔有一席之地，並由活躍的社群提供支持，該社群已建立超過 5,000 個節點（Nodes）和流程（Flows），幫助您快速開始。

 

開始使用

Node-RED 基於 Node.js 構建，充分利用其事件驅動、非阻塞模型。這使其非常適合在 Raspberry Pi 等低成本硬體上的網路邊緣運行，也可以在雲端運行。

在本機運行

  • 開始使用
  • Docker

在設備上運行

  • Raspberry Pi
  • FlowFuse 設備代理程式 (Device Agent)
  • BeagleBone Black
  • 與 Arduino 互動
  • Android

在雲端運行

  • FlowFuse
  • Amazon Web Services
  • Microsoft Azure
